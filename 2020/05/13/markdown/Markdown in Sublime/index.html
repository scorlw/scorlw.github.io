Markdown in Sublime

scorlw 发布于 2020-05-13

找了挺多的Markdown软件的，网易云笔记，typora，Tea……各有各的好处，但是偶然看到sublime text3也有Markdown的插件，因为平时自己用sublime text多一些，所以安装了对应的插件，感觉效果还不错，安装过程记录一下。
尝试了四个插件，有一个不大适合我，所以后来只是安装了三个。

1. MarkdownEditing

可以说是一个基础，可以高亮显示Markdown的语法，也可以自动匹配星号、下划线及反引号，总之是简化了一些操作，可以安装。

2. MarkdownLivePreview（不推荐）

可以实时预览Markdown文件，安装之后要经过 Preferences -> Package Settings -> MarkdownLivePreview -> Settings 的设置的右侧加一条“ "markdown_live_preview_on_open": true, ”，重启sublime即可使用。
存在问题
实际预览结果并不是很美观，不能横向滚动，而且没有自动换行，所以有些文字是看不到的，就很难受。

3. MarkdownPreview

可以在浏览器中预览Markdown文件。

操作方法一：

  1. 通过快捷键*​ Ctrl+Shift+P *​呼出命令面板；

  2. 输入​ Markdown Preview: Preview in Browser （可以通过简写mdp快速找到）

  3. 有两种模式可以选择：​github *​和 *​markdown​ 。github **是利用GitHub的在线API进行解析，支持在线资源的预览，速度取决于网速；​markdown**​ 利用的是本地的API，可能不支持在线资源的预览。

    操作方法二：

    可以通过自定义快捷键的方法快速打开。
    方法是在 打开的文件的右侧栏的中括号中添加一行代码：
    { "keys": ["alt+m"], "command": "markdown_preview", "args": {"target": "browser", "parser":"markdown"} }
    其中的 ​alt+m **​和​ parser : markdown**​ 也可以自定义。

    4. LiveReload

    可以配合MarkdownPreview来使用，作用是可以自动刷新网页。
    安装好后通过快捷键*​ Ctrl+Shift+P *​呼出命令面板，输入, 回车, 选择 或者，后者没有延迟。
    这样每次保存一下文件，网页就会同步刷新一下。
    有个小缺点是网页不会自动同步到当前的编辑位置。
