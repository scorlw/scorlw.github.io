1 Markdown的基本语法

1. 标题

输入#+空格就可以自动生成一级标题，同理的，几个 # 就是几级标题，Typora中限制标题级数最多到六级（大部分都是到六级）。

2. 粗体、斜体

斜体 是在输入文字的两端加上一个星号 *，而 粗体 则是加上两个星号，粗斜体的话就加三个。

星号与文字之间不需要空格。

3. 分割线

另起一行，输入三个星号或者三个减号就可以生成分割线。

5.表格

表格的插入还是比较麻烦的，但是 Typora 可以右键插入一个表格（当然也可以插入其他的元素），这样就可以在图形化的表格中进行编辑了。

表头1 表头2 表头3

6. emoji

在 Typora 中可以快速插入emoji表情（其他markdown软件可能不支持），输入:+emoji描述就可以快速插入emoji，例如输入:star:可以生成:star:。
