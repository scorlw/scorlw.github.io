c_str函数

c++

语法:

const char *c_str();

c_str()函数返回一个指向正规C字符串的指针常量, 内容与本string串相同.

这是为了与c语言兼容，在c语言中没有string类型，故必须通过string类对象的成员函数c_str()把string 对象转换成c中的字符串样式。

  1. c_str是一个内容为字符串指向字符数组的临时指针；

  2. c_str返回的是一个可读不可改的常指针；

注意：一定要使用strcpy()函数 等来操作方法c_str()返回的指针
