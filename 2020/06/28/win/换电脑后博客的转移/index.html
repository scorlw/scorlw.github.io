换电脑后博客的转移

前提：新电脑已经安装并配置好git、npm与hexo。

一、基本配置

  1. 在新电脑里新建文件夹，并在该文件夹下打开cmd；
  2. 输入hexo init进行博客初始化；
  3. 输入hexo g编译，输入hexo s打开本地预览
  4. 在浏览器里打开http://localhost:4000，看到的是hexo的默认主界面，则初始化成功。
  5. 输入npm install hexo-deployer-git --save安装依赖库。
